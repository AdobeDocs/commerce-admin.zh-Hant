# 取消指派來源

>[!NOTE]
>
>取消指定來源會清除所有數量資料。 重新指定來源可能會造成可銷售數量、預留及暫緩訂單的問題。 建議您先處理所有訂單，再取消指定來源或進行其他來源變更。 您也可以使用新指令來偵測和更新預留。 如需詳細資訊，請參閱[[!DNL Inventory Management] CLI參考](../inventory-management/cli.md)。
