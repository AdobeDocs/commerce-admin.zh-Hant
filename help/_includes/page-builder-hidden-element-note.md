# [!DNL Page Builder]個隱藏元素

>[!NOTE]
>
>隱藏的元素會儲存在資料庫中，且對客戶不可見。 不過，搜尋引擎和其他對您的網站進行編目的機器人可看見這些元素。 如果透過具有不可見屬性的API呼叫提出請求，這些區段也會當作內容的一部分傳回，除非您將其從階段中移除。
